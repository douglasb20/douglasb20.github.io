BackupTec

Sistema de backup automatizado de bancos de dados com armazenamento seguro no Google Drive.

Sobre o aplicativo

O BackupTec é um aplicativo Windows que realiza backups periódicos de bancos de dados e os envia automaticamente para o Google Drive do usuário.
